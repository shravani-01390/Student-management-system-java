# Student Management System (Java)

## Description
This is a console-based Student Management System developed using Java.
It uses OOP principles and File Handling to store student data.

## Technologies Used
- Java
- OOP Concepts
- ArrayList
- File Handling (BufferedReader, FileWriter)

## Features
- Add Student
- Display Students
- Search Student by Roll Number
- Sort Students by Marks
- Persistent Data Storage

## How to Run
1. Compile the program:
   javac *.java
2. Run the program:
   java Main
